OSD Group Project 2026

 Open Source Software Development (OSD) Group Project
Project Title

Group Portfolio Website

Project Overview

The Group Portfolio Website is a collaborative web development project designed to showcase the profiles, skills, and contributions of all eighteen (18) members of the Open Source Software Development (OSD) group. The project demonstrates teamwork, version control practices, and collaborative software development using Git and GitHub.

 How to Run the Project

To run the project locally:

1. Download or clone the project repository.
2. Locate the `index.html` file.
3. Open the file using any modern web browser such as Chrome, Firefox, Edge, or Safari.

Technology Stack

The project was developed using the following technologies:
HTML5 – Structure and content of the website.
CSS3 – Styling and layout design.
JavaScript – Interactive features and functionality.
