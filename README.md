 How to Contribute

1. Fork this repository
2. Create a new branch: `git checkout -b add-your-name`
3. Add your profile card in `script.js`
4. Commit and push your changes
5. Open a Pull Request to the main repository

 Tech Stack
- HTML5
- CSS3
- JavaScript
