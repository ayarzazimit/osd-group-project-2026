OSD Group Project 2026
Group Portfolio Website
A collaborative web development project by the Open Source Software Development (OSD) Group. This portfolio website showcases the profiles, skills, and contributions of all 18 members of the group.
The project demonstrates effective teamwork, proper Git & GitHub workflows (branching, committing, pull requests), and modern web development practices.
Live Demo
🌐 View the website here:
https://ayarzazimit.github.io/osd-group-project-2026/

Project Overview
The Group Portfolio Website serves as a digital showcase for the OSD group members. Each member contributed unique features and improvements while maintaining clean version control practices.
Key features include:
Responsive design with modern UI/UX
Dark/Light mode toggle
Interactive profile cards with circular images and hover effects
Smooth animations and scroll-triggered reveals
Skills section
Contact form section
Professional navigation bar
Fully responsive across devices
Technology Stack

HTML5 – Semantic structure and content
CSS3 – Styling, animations, responsiveness, and dark mode
JavaScript – Interactivity, dark mode, dynamic profile rendering, and animations
How to Run Locally
git clone https://github.com/ayarzazimit/osd-group-project-2026.git
Navigate to the project folder.
Open index.html in any modern web browser (Chrome, Firefox, Edge, Safari).

No installation or build tools required.
